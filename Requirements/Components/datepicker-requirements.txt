================================================================================
          PROJECT: PERMACULTURE ADMINISTRATIVE PORTAL
          TECHNICAL STANDARD: THE BILINGUAL FLATPICKR DATEPICKER
================================================================================

This document outlines the standard for date and time selection across the
entire application. All forms requiring temporal input MUST follow this standard.

1. CORE TECHNOLOGY:
-------------------
We have migrated to "Flatpickr" as the enterprise standard for temporal selection.
- Library: flatpickr
- Wrapper: resources/js/Components/DateInput.vue
- Why: Lightweight, zero dependencies (native JS), excellent bilingual support, 
       and perfect integration with the project's 42px height standard.

2. VISUAL & UI STANDARDS (PERMACULTURE THEME):
---------------------------------------------
- Standard Height: MUST be exactly 42px to match Select2 and text inputs.
- Primary Color: #198754 (Success Green).
- Typography: 0.95rem font size for the input field.
- Icons: Must use Boxicons (e.g., bx-calendar, bx-time) as input group addons.
- Border: 2px border width on focus with #198754 color.

3. COMPONENT ARCHITECTURE:
--------------------------
Use the specialized component for bilingual forms:
`resources/js/Components/DateInput.vue` (Standard)
`resources/js/Components/BilingualDateInput.vue` (For mixed forms)

Standard Formats:
- Date Only: "Y-m-d" (Backend), display formatted via Flatpickr.
- Time Only: "H:i" (24-hour format standard for schedules).

4. BILINGUAL SUPPORT (EN/BN):
----------------------------
The datepicker MUST respond to the application's locale.
- Localized Month/Day names must be loaded dynamically.
- Input fields should support Bangla numeral display/input where possible 
  to maintain the "Bangla-First" UX.

5. USAGE & ACCESSIBILITY:
-------------------------
- Allow Input: Users must be able to type dates manually (altInput: true).
- Clear Function: Standard X button (styled like Select2) should be available 
  for optional date fields.
- Placement: Teleport to body (static: false) to prevent clipping in 
  scrollable cards or modals.

6. INSTALLATION:
----------------
npm install flatpickr
